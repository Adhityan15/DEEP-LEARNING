# DEEP-LEARNING
A comprehensive Deep Learning laboratory repository containing 10–12 experiments covering CNNs, U-Net, Image Processing, Optimization, and Model Evaluation using PyTorch
Deep Learning Laboratory

This repository contains a structured collection of Deep Learning laboratory experiments implemented using PyTorch, NumPy, Matplotlib, and related libraries.

The experiments cover core deep learning concepts including neural networks, convolutional neural networks (CNN), image processing, segmentation, optimization techniques, and model evaluation.

🎯 Objectives

Understand fundamentals of Deep Learning

Implement CNN architectures from scratch

Train and evaluate neural networks

Perform image segmentation using U-Net

Apply optimization algorithms

Visualize model predictions and performance metrics

🚀 How to Run

Clone the repository:

git clone https://github.com/yourusername/Deep-Learning-Laboratory.git

Install dependencies:

pip install torch torchvision numpy matplotlib opencv-python

Run any experiment:

python code.py
